# moonbit-pixelkit 项目申报书

## 基本信息

- 项目名称：moonbit-pixelkit
- GitHub 仓库链接：https://github.com/zhenghao493-netizen/moonbit-pixelkit
- 项目方向：MoonBit 2D 网格地图解析与 Pathfinding 工具库
- 是否为移植项目：否，原创项目

## 项目简介

moonbit-pixelkit 是一个面向 2D 像素游戏、网格地图应用和算法教学场景的 MoonBit 开源工具库，计划提供轻量、可测试、易集成的地图解析与路径搜索能力。

项目希望解决小型游戏 Demo、关卡原型、战棋/迷宫/Roguelike 等网格类应用中常见的基础需求：开发者可以用 ASCII 文本地图或 CSV tile 数据描述地图，并快速完成地图解析、障碍判断、移动代价查询、可达区域计算和最短路径搜索。

项目将以 MoonBit 为主要实现语言，采用 MIT License 开源，提供清晰 README、可运行示例、单元测试和 CI，后续计划发布到 mooncakes.io，作为 MoonBit 游戏与算法生态中的基础组件。

## 核心功能范围

- 提供统一的 2D 网格地图数据结构，支持宽高、坐标、tile id、墙体、空地和移动代价等基础信息；
- 支持 ASCII 文本地图解析，便于快速编写迷宫、关卡原型和算法测试用例；
- 支持 CSV tile 数据解析，便于与常见 2D tile map 工作流衔接；
- 提供地图查询接口，包括坐标越界判断、tile 查询、可通行性判断和移动代价查询；
- 实现 BFS 可达区域搜索，用于计算角色在固定步数或固定代价内可到达的位置；
- 实现 A* 最短路径搜索，支持四方向和八方向邻居策略；
- 支持加权地形，用于表达道路、草地、水域等不同移动成本；
- 提供示例程序，覆盖 ASCII 迷宫寻路、加权地图寻路和游戏循环接入方式；
- 提供单元测试，覆盖地图解析、非法输入、不可达路径、边界坐标、四方向/八方向搜索和加权路径选择；
- 提供 README 使用说明、CI 检查和 MoonBit 包发布准备。

## 原创或参考说明

本项目为原创 MoonBit 开源项目，不是对某个已有项目的直接移植。项目中的 BFS、A* 等算法属于通用路径搜索算法，具体 API 设计、数据结构组织、示例代码、测试用例和 MoonBit 实现将由本项目独立完成。

如后续在接口设计、示例组织或测试数据中参考其他开源项目，将在 README 中明确列出参考来源、链接和许可证，并确保代码来源清晰、许可证兼容。
